= FMS Bingo System

== Spec

* Requires Ruby ~> 1.9
* Chrome browser

== Install

    $ git clone git://github.com/udzura/jqbingo.git
    $ cd jqbingo
    $ bundle

== Booting

    $ rackup -p 3000

Then open http://localhost:3000/bingo.html

== UI Keybind

* "r" button, to start/stop bingo
* "F5" button, to reflesh
